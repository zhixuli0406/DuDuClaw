## 一、昨日完成事項

### 🚀 v1.15.0 發布（今日）— Cross-Platform PTY Pool + Worker
這是本週最重要的里程碑。背景：Anthropic 在 2026 中期封鎖 OAuth 訂閱帳號使用 `claude -p`，v1.15.0 通過真實 PTY 驅動互動式 `claude` REPL 作為解法。

**核心產出**：
- 新 crate `duduclaw-cli-runtime`（portable-pty 跨平台包裝，支援 macOS/Linux/Windows）
- 新 crate `duduclaw-cli-worker`（獨立 worker binary，localhost HTTP+JSON-RPC，Bearer token 認證）
- Gateway 整合：PTY 路由、worker supervisor、runtime status endpoint
- Phase 8 生產可觀測性：12 個 Prometheus metrics（pool 取得/驅逐/調用統計）
- Kill switch 設計：per-agent `pty_pool_enabled`（預設 OFF）+ 全域 env-var 緊急關閉
- Darwin x64 binary 已打包：`artifacts/duduclaw-darwin-x64.tar.gz`

### v1.14.0（2026-05-14）— RFC-23 Redaction Pipeline（上週完成）
- AES-256-GCM 加密 SQLite vault
- 5 層啟用/停用解析器（channel force_on → env+CLI → agent.toml → config.toml）
- MCP 層整合（工具參數 redact/restore，LLM 永遠看不到原始內部資料）
- 98 unit tests + 11 E2E 整合測試
