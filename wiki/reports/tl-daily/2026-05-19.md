## 六、技術決策（W24 Day 1）

### TD-2026-0519-01：Anthropic Dreaming vs DuDuClaw Session Dreaming — 架構響應
**背景**：PM 2026-05-19 報告指出，Anthropic 於 2026-05-07 正式公告 Claude Managed Agents 三項新功能：
1. **Dreaming**：自動回顧 agent sessions，萃取 pattern，更新記憶，支援「自動更新」或「人工審核後更新」兩種模式
2. **Outcomes**：獨立 grader 評估輸出，失敗自動重試 + webhook
3. **Multiagent Orchestration**：lead agent 分解任務，並行子 agent

DuDuClaw 於 2026-05-15 提出的 Session Dreaming 方向與 Anthropic Dreaming 高度重疊，**方向正確，但需差異化**。

**TL 裁定**：
- ENG-MEMORY 在 Tiered Memory SDD 中**納入 Dreaming-aware 架構設計**（記憶層需支援 Dreaming-style 模式萃取）
- PM 本週提交「Anthropic Dreaming vs DuDuClaw Dreaming 差異化策略」（定位 DuDuClaw 的獨特競爭優勢）
- TL 暫不採用 Anthropic Managed Agents 原生 Dreaming（避免 vendor lock-in），保持自主架構優先

### TD-2026-0519-02：Mem0 競品威脅響應
**背景**：Mem0 新演算法（+29.6 pts 時序、+23.1 pts 多跳）直接壓縮 DuDuClaw ENG-MEMORY 的相對競爭優勢。

- ENG-MEMORY SDD 截止前需納入 Mem0 新演算法比較分析
