## 核心功能

Ruflo 是為 Claude 量身打造的多 Agent 協作平台，支援部署 100+ 專責 AI Agent 組成 swarm 協同運作。平台整合 SONA 神經模式實現自學習架構，並以 HNSW 索引的 AgentDB 向量記憶體加速搜索（聲稱達 150x–12,500x）。提供 Web UI（flo.ruv.io）與目標規劃器（goal.ruv.io，使用 GOAP A* 演算法），以及 32 個原生 Claude Code 插件加上 marketplace 生態系統。

## 亮點與創新點

- **Agent Federation**: 零信任安全架構下的跨機器 Agent 安全協作
- **SONA 神經模式 + ReasoningBank**: 持久化推理記憶，Agent 可從過去決策中學習
- **12 個自動觸發背景 worker**: 免配置即可運行的後台任務處理
- **AIDefence + CVE 修復**: 內建企業級安全掃描與自動漏洞修補模組
- **MCP 並行工具呼叫**: 透過 MCP server 整合 Claude Code 生態系
