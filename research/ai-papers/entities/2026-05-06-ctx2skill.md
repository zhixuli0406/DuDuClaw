# From Context to Skills: Can Language Models Learn from Context Skillfully?
**Ctx2Skill：語言模型能否從上下文中自主習得技能？**

## 基本資訊

- **作者：** Shuzheng Si, Haozhe Zhao, Yu Lei, Qingyi Wang, Dingwei Chen, Zhitong Wang, Zhenhailong Wang, Kangyang Luo, Zheng Wang, Gang Chen, Fanchao Qi, Minjia Zhang, Maosong Sun（共 13 位）
- **機構：** 清華大學等（依作者群推斷）
- **發表日期：** 2026-04-30（修訂於 2026-05-03）
- **來源：** arXiv cs.CL
- **論文連結：** https://arxiv.org/abs/2604.27660

## 核心貢獻

本文提出 **Ctx2Skill** 框架，讓語言模型在無需人工標注或外部回饋的情況下，從提供的上下文中自主發現並習得可複用技能（reusable skills）。系統核心是一個**多智能體自我博弈迴圈**（multi-agent self-play loop），由競爭與協作的智能體共同探索、精煉並評估技能，搭配重播機制防止技能退化。

## 方法摘要

- **多智能體自我博弈（Multi-Agent Self-Play）**：競爭型智能體發現技能候選；協作型智能體精煉與驗證
- **自主技能發現**：不依賴人工標注，模型自行從上下文歸納可複用的解題策略
- **技能重播機制（Replay Mechanism）**：防止隨訓練進行發生的技能遺忘（skill deterioration）
- 在四個 context learning benchmark 上驗證，跨不同語言模型架構均有穩定提升

## 實驗結果

在四個 context learning 基準測試中，Ctx2Skill 相較 baseline 在不同語言模型架構上均顯示一致性提升，顯示方法的通用性。

## 重要性評估

In-context learning（ICL）一直是 LLM 最重要的能力之一，但現有研究多聚焦在如何**利用**上下文而非如何從中**學習技能**。Ctx2Skill 開創了一個新研究方向：讓模型自主將上下文提煉為結構化技能，這對長期 agent 記憶、持續學習與少樣本泛化均有重要意義。在 HuggingFace Daily Papers 獲得 120 upvotes（2026-05-05 排名第二）。
