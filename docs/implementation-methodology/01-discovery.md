# Phase 1: Discovery — 需求訪談與場景盤點

## 目標

在 1-2 次會議中完成客戶需求收集，產出明確的 PoC 範圍定義。

### 現況了解

1. **目前客服/業務流程如何運作？**
   - 使用哪些通訊工具？（LINE OA / Telegram / 電話 / Email）
   - 每天大約處理多少筆客戶詢問？
   - 回覆平均花多少時間？
   - 有幾位人員負責回覆？

2. **最大的痛點是什麼？**
   - □ 回覆不夠即時（漏接、延遲）
   - □ 人力成本太高
   - □ 重複性問題太多
   - □ 非營業時間無人回覆
   - □ 資訊散落各處，不一致
   - □ 其他: ___

3. **曾經嘗試過什麼解決方案？**
   - 用過哪些工具？結果如何？
   - 為什麼放棄或不滿意？
