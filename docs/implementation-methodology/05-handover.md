# Phase 5: Handover — 結案交付與維護

## 目標

完成正式交付，建立長期維護關係。

## 結案交付物清單

### 技術文件

- [ ] Agent 配置說明書
  - agent.toml 各欄位說明
  - SOUL.md 結構與修改指南
  - CONTRACT.toml 邊界說明
- [ ] 系統架構圖（客戶環境）
- [ ] 頻道設定文件（LINE OA / Telegram / Discord）
- [ ] 備份與恢復 SOP
- [ ] 故障排除指南（常見問題 + `duduclaw doctor` 使用）

### 帳號與權限

- [ ] Dashboard 登入憑證
- [ ] LINE OA 管理權限確認
- [ ] DuDuClaw 授權金鑰
- [ ] API Key 管理說明

### 教育訓練

- [ ] Dashboard 操作訓練（1-2 hr）
  - 基本導覽
  - Agent 狀態監控
  - 記憶體更新流程
  - 費用監控
- [ ] SOUL.md 維護訓練（1 hr）
  - 如何新增 FAQ
  - 如何調整語氣
  - 如何更新促銷/價格資訊
- [ ] 緊急應對訓練（30 min）
  - Agent 異常時如何暫停
  - 如何切回人工客服
  - 如何聯繫技術支援

### 數據報告

- [ ] 完整陪跑期數據報告
- [ ] ROI 分析（成本節省量化）
- [ ] 建議事項（後續優化方向）

## 維護合約範本

```
DuDuClaw 維護合約

甲方：___（客戶）
乙方：___（DuDuClaw）

一、維護範圍
  1. Agent 正常運行保障
  2. SOUL.md 季度檢視與建議
  3. DuDuClaw 版本更新
  4. 安全補丁即時部署
  5. 技術諮詢（Email / LINE，工作日 9-18）

二、維護期間
  自 ___ 年 ___ 月 ___ 日起，為期 ___ 個月

三、費用
  月費 NT$ ___（含稅）
  付款方式：□ 月繳 □ 季繳（95折）□ 年繳（9折）

四、服務等級（SLA）
  - 一般問題回覆：24 小時內
  - 系統異常回覆：4 小時內
  - 緊急事故回覆：1 小時內
  - 系統可用性目標：99.5%（不含計畫性維護）

五、不包含項目
  - 新功能開發（另行報價）
  - 硬體維護
  - 第三方服務費用（LINE OA、API 費用）

六、自動續約
  合約到期前 30 天未書面通知解約，自動續約 ___ 個月
```
