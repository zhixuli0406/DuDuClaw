# DuDuClaw 內容分層策略

> 定義公開、半公開、付費內容的邊界，確保推廣與商業化一致。

---

## 1. 公開內容（What + Why）

任何人可自由取用，用於吸引流量與建立信任。

- 安裝教學：完整 `brew install` + `duduclaw onboard` 流程
- 單 Agent + 單頻道 Demo
- 架構原理、設計理念文章
- Community 版全部功能操作教學
- GitHub README、CLAUDE.md、公開文件

## 2. 半公開內容（How — 付費教育）

體驗營 / 實戰班限定，用於轉換付費學員。

- 多頻道整合實戰（LINE + Telegram + Discord 同時管理）
- Odoo ERP 整合 step-by-step
- Evolution Engine 參數調校技巧
- 踩坑經驗 + 排錯 SOP
- 本地推論最佳化（模型選擇、量化策略、硬體配置）

## 3. 付費專屬內容（How Well — 導入服務）

僅限付費客戶，用於產生持續收入。

- 客製化 SOUL.md 人格設計
- CONTRACT.toml 行為邊界顧問
- 持續調校 + 月度優化報告
- SLA + 緊急支援
- 產業專用 GGUF 模型（含在維護合約中）

---

## YouTube 內容規則

- 每支影片結尾固定 CTA：LINE OA QR Code + 官網連結
- Pro/Enterprise 功能：只展示效果畫面，不教完整設定步驟
- Evolution 演化：展示 SOUL.md 自動改善的前後對比，不教參數
- Odoo 整合：展示「詢價 → 自動查價目表 → 回覆」流程，不教設定

## 客戶黏著機制

- **自動更新**：付費版 `duduclaw update` 即時取得最新版 + 安全補丁
- **模型庫訂閱**：持續更新產業專用 GGUF 模型（含在維護合約中）
- **SOUL.md 模板市集**：經實戰驗證的 Agent 人格，持續新增
- **數據飛輪提醒**：Dashboard 顯示「您的 Agent 已累積 N 筆記憶，演化 M 次」
